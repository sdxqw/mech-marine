> **Project Task:**
> Analyze the existing website/project first. Do not start coding immediately.
>
> 1. Review the current structure, content, and purpose of the website.
> 2. Identify weaknesses in layout, UX, pricing presentation, and positioning.
> 3. Briefly outline the improvement strategy.
> 4. Only after that, start implementing the improved version.
>
> ---
>
> **Technical Stack Requirements:**
>
> * Use **Next.js (App Router)**
> * Use **ShadCN UI** for components
> * Use **Tailwind CSS** for styling
> * Use **Framer Motion** for modern, smooth animations (subtle, professional, not excessive)
> * Fully responsive design (mobile-first approach)
>
> ---
>
> **Services to Present Clearly (with separate pricing plans):**
>
> * Website development (Next.js + ShadCN + Tailwind)
> * Hosting setup using Docker
> * CI/CD with GitHub Actions
> * Domain configuration via Namecheap
> * Google Analytics setup
> * Google Search Console setup
> * Email form integration
>
> These must be presented as structured service packages with clear differences and pricing tiers.
>
> ---
>
> **Business Requirements:**
>
> * Analyze competitors in the same market and adjust pricing strategically.
> * Present pricing in a competitive but profitable way.
> * Clearly explain what is included in each plan.
> * Highlight value, not just features.
>
> ---
>
> **Design Requirements:**
>
> * Use modern, meaningful color palettes (professional and trustworthy).
> * Clean typography and strong visual hierarchy.
> * Modern layout with clear sections:
>
>   * Hero section
>   * Services
>   * Pricing plans
>   * Process / How it works
>   * Contact / Call to action
> * Smooth scroll animations using Framer Motion.
> * Optimized for performance and SEO best practices.
>
> ---
>
> **Important:**
>
> * Write clean, scalable, production-ready code.
> * Keep components modular and reusable.
> * Avoid unnecessary complexity.
> * Focus on clarity, performance, and conversion optimization.
